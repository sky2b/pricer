AddPricer: data-mining components

Price your products wisely

AddPricer is a general purpose pricer. It is a command line data processing application driven by *.yaml makefiles. The idea is similar to the one used in C make (or ant, grunt, etc). Just by editing XML configuration you may instruct the application to read data from *.csv files, export results to MySQL database, perform HTTP requests, etc.

Know your competitors

Manually checking for competitor's prices can be pretty tedious! AddPricer has an embedded web harvester - a data mining tool, which could be instructed to fetch any web page, parse HTML and extract the data needed. Right out of the box it comes with examples of how to extract Amazon listings, but could be used for any other web site.
